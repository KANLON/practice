&emsp;&emsp;本项目主要用来记录自己尝试使用Google的开源OCR（光学字符识别(OCR,Optical Character Recognition)）项目来自己训练字体及识别图片文字的过程的文件。做过之前才发现自己训练好一个字体真的非常非常麻烦，建议如果识别多的文字还是使用百度AI中的API比较好（虽然它识别多了要收费），如果识别的文字比较少或不是打印文字可以自己训练字体。
